## oms install docker

Install Docker on a remote host

### Synopsis

Install Docker (if not already present) on a remote host accessed via SSH.

```
oms install docker [flags]
```

### Options

```
  -h, --help              help for docker
      --ssh-host string   Remote host IP or hostname (required)
      --ssh-port int      SSH port on the remote host (default 22)
      --ssh-user string   SSH username (default "root")
```
